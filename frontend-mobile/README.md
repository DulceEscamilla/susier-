# SUSIER Mobile App

## 📱 Tecnologías

- React Native + Expo
- TypeScript
- React Navigation
- Expo Camera & Barcode Scanner
- Expo Location
- Zustand para state management

## 📦 Instalación

```bash
cd frontend-mobile
npm install
npm start
```

## 🏃 Ejecutar

```bash
# iOS
npm run ios

# Android
npm run android

# Web
npm run web
```

## ✨ Características

- ✅ Autenticación segura
- ✅ Lector de códigos QR
- ✅ Acceso a cámara
- ✅ Geolocalización
- ✅ Diseño responsivo
- ✅ Modo oscuro
